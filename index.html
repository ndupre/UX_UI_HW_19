My Skills

User Research

I know how to plan and conduct User Research, User testing and how to implement Surveys to base my designs on quantitative and qualitative information to identify the problem and ideate solutions.

Ideation

Having my user persona on hand and the problem identified, I generate possible solutions and prioritize them depending on their impact and faceability. With this I can estate the user journey using different methods as storyboards, empathy maps or user journey maps.

Prototyping

I’m skilled defining user flows, often taking them to quick paper prototypes and Lo-fidelity wireframes to test and iterate till defining Hi-fidelity mocks.
